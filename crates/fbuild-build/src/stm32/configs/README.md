# STM32 MCU Configs

JSON configuration files for STM32 MCU families.

Each file contains compiler flags, linker flags, and profiles for a specific
ARM Cortex-M variant used by STM32 microcontrollers.

- `stm32f1.json` - STM32F1xx (Cortex-M3, no FPU)
- `stm32f4.json` - STM32F4xx (Cortex-M4F, hardware FPU)
- `stm32h7.json` - STM32H7xx (Cortex-M7, double-precision FPU)
